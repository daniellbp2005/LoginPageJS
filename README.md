# Login e Cadastro

Aplicação web de autenticação com cadastro e login de usuários, com persistência de dados via localStorage.

## Tecnologias
- HTML
- CSS
- JavaScript

## Funcionalidades
- Cadastro de novos usuários
- Login com validação de email e senha
- Verificação de conta duplicada
- Persistência de dados com localStorage

## Como Rodar
1. Clone o repositório: `git clone <url>`
2. Abra a pasta do projeto
3. Abra o arquivo `index.html` no navegador
